Copy of Charles Proxy로 네트워크 조정하기

이 페이지에서는 Charles를 사용하여 Network Link Conditioner 를 대체하는 방법을 알아보겠습니다.

  • Charles를 실행하고 네트워크를 조정할 테스트 기기를 연결합니다.
  • 상단 메뉴 → Proxy → Throttle Settings 메뉴로 이동합니다.
  • Enable Throttling을 체크하여 활성화합니다.
  • Bandwidth, Utilisation, Latency 등 네트워크 옵션을 설정하여 preset를 만들 수 있습니다.
  • 이미 만들어져있는 preset을 활용해도 됩니다. (56 kbps ~ 4G)
  • OK를 누르면 설정이 완료됩니다.
  • 거북이 모양의 아이콘(🐢)을 클릭하면 throttling이 시작됩니다.

네트워크 설정이 잘 적용이 되었는지 확인하기위해서, 56 kbps Modem으로 preset을 설정하고 fast.com 에서 인터넷 속도를 비교해보겠습니다.

  • Throttling 미적용 → 43 mbps

  • Throttling 적용 → 28 kbps

인터넷 속도가 48 mbps에서 28 kbps로 느려진 것을 확인할 수 있습니다.

Throttle setting이 잘 적용되었습니다. 🎉
