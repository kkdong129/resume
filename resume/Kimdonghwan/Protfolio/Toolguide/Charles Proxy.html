Charles Proxy

개요

Charles Proxy는 연결된 기기에 모든 HTTP 및 SSL / HTTPS 트래픽을 볼 수 있는 디버깅 툴입니다.

(https://www.charlesproxy.com/)

설치 방법

  1. 사용 중인 컴퓨터에 https://www.charlesproxy.com/download/ 접속하여 Charles를 설치합니다.
  1. Charles가 트래픽을 볼 수 있도록 Charles를 설치한 컴퓨터와 테스트에 사용할 Android와 iPhone을 모두 같은 네트워크(Wifi)로 연결합니다.
  1. Charles를 실행하고 상단 메뉴에 Help → Local IP Adress를 클릭해서 현재 연결된 컴퓨터 네트워크의 아이피 주소를 확인합니다.
  1. 상단 메뉴에 Proxy → Proxy Settings에서 HTTP Proxy 포트를 8888이나 임의로 입력합니다. (기본 포트는 8888입니다.)
  1. Proxy → SSL Proxying Settings → Add로 모든 호스트와 포트에서 모니터링할 수 있도록 와일드 카드(*)를 입력해서 추가합니다.
  1. iPhone: wifi 설정 → 연결된 네트워크 오른쪽에 i 버튼 → HTTP 프록시 / 프록시 구성 → 수동, 서버에 Local IP Adress 입력, 포트(기본 8888)를 입력해줍니다.
  1. Android: wifi 설정 → 연결된 네트워크 오른쪽에 i 버튼 or 고급 설정 → 프록시 수동 → 호스트 이름에 Local IP Adress 입력, 포트(기본 8888)를 입력해줍니다.
  1. 성공적으로 연결되면, Charles에 얼럿이 나옵니다. Allow를 클릭해줍니다.
  1. PC와 모바일 폰이 연결된 상태에서 Charles 인증서를 설치해줍니다.
  1. Android, iOS : http://charlesproxy.com/getssl 접속하여 인증서 다운로드 또는 프로파일 설치 PC : 상단 메뉴 → Help → SSL Proxying → Install Charles Root Certificate → 키체인 → 정보가져오기 → 항상 신뢰로 변경
  1. 연결이 되면 Structure와 Sequence에 모니터링(기록)이 시작됩니다. Proxy → macOS Proxy에 체크를 해제해주면, 연결된 기기만 모니터링하게됩니다.

이제 Charles를 사용할 준비가 되었습니다. 🎉

연결된 기기를 사용하면 Charles proxy에 기록되기 시작합니다.
