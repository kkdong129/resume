libimobiledevice 명령어로 iPhone 로그 파일 저장하기

iPhone은 ADB(Android Debug Bridge)와 같은 툴이 없을까하고 찾고 작성하게 되었습니다. 다양한 툴이 있지만, 이 페이지에서는 libimobiledevice tool을 설치하고 사용해서 직접 로그파일까지 저장해보겠습니다.

libimobiledevice는 기본적으로 iOS 장치와 상호 작용할 수있는 오픈 소스, 크로스 플랫폼 소프트웨어 프로토콜 라이브러리 및 도구입니다. ADB와 같이 Mac과 iPhone을 케이블로 연결해서 터미널에서 몇가지 명령어를 사용할 수 있습니다.

설치 방법

공식홈페이지에 다양한 방법이 있지만 저는 brew를 사용해서 설치하려고합니다. 터미널을 실행하고 아래 명령어를 입력하면 설치됩니다.

$ brew install libimobiledevice

로그 파일 저장하기

터미널을 실행하고 Mac과 iPhone을 케이블로 연결해줍니다. 터미널에 아래와 같이 명령어를 입력해줍니다. 저장 위치와 파일명을 입력할 수 있습니다. 명령어를 입력하고나서는 Control + C로 실행을 중지합니다.

예시: 바탕화면에 test라는 로그 파일을 저장하는 경우

$ idevicesyslog > /Users/damon/Desktop/test.log
#exit: Control + C

아래와 같은 명령어를 사용하면 특정 앱의 로그만 저장할 수 있습니다.

$ idevicesyslog | grep "App Name" > /Users/damon/Desktop/test.log
#idevicesyslog -m "App Name" > /Users/damon/Desktop/test.log
#exit: Control + C

예시: 바탕화면에 test라는 Youtube 앱의 로그 파일을 저장하는 경우

$ idevicesyslog | grep "Youtube" > /Users/damon/Desktop/test.log
#idevicesyslog -m "Azar" > /Users/damon/Desktop/test.log
#exit: Control + C

이외에도 몇가지 명령어가 있는데 업무에 사용할 수 있는 명령어가 있는지 확인이 필요합니다.
